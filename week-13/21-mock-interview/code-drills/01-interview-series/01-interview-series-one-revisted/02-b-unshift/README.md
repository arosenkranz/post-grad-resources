## Unshift

## Instructions
Write a function that takes an element and an array  as arguments.

The function sould insert the element at the beginning of the array A i.e at index 0. 

## Example
Test Case 1:  [1,30,13,3], 9
Expected Output: [9,1,30,13,3]

Test Case 2:  [], 4
Expected Output: [4]

## Optimal Time
O(n)
